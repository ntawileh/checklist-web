About

checklist provides an interface to Google Tasks™ via a very simple and intuitive Chrome extension. Lists of tasks and the ability to quickly add a task are a keyboard shortcut away.

  • fast access to tasks from Chrome
  • ability to add a page, or text from a page, as a task directly via right-click menu and keyboard shortcuts
  • great for use with gmail to make a task out of an email; adding a page as a task stores the URL of that page for quick access later
  • support for multiple lists and consolidated view of all lists

checklist browser extension communicates directly with Google Tasks™ via Google APIs and does not otherwise persist or store your data anywhere else.

Usage

Examples of things you can add...

buy milk will add a task with the title 'buy milk' and schedule it for today
buy milk by next wed will add a task with the title 'buy milk' and schedule it for next Wednesday
buy milk by tomorrow will add a task with the title 'buy milk' and schedule it for tomorrow
buy milk by Oct 2 get it? will add a task with the title 'buy milk' and schedule for October 2nd
read thispage by tomorrow thispage is a special keyword. it will embed the title of the currently open tab and the URL of the tab in the task
thispage by fri create a task with title being the open tab's title and include the URL in the task. task will be scheduled for friday
by fri if there is nothing before 'by', it assumes you are referring to the current page (equivalent to previous row)

You can also add tasks using the context/right click menu on any page. If you have any text selected, that text will be the title of your new task. If there is no text selected, the name of the opened web page will be the title. Either way, the URL of that page is saved in the task notes and can be opened later.

Keyboard shortcuts

There are two keyboard shortcuts that can be defined.

  • One just opens the extension popup.
  • The other will 'quick task' the current page as a task for later today (equivalent of thispage by today).

The shortcuts can be configured from the Chrome Extensions page:

(chrome://extensions/shortcuts)

Multiple Lists

Support for multiple lists is available. You can add/remove lists from the options page.

  • In the popup menu, you will see a select option on top that allows you to switch to different lists. You will only see this if you have more the one list.
  • One special selection is "All Tasks", which will show all tasks in all your lists.
  • A new task will be added to the list that is currently selected. If "All Tasks" is selected, the task is added to your default list.
  • You can always move a task to another list using the task edit popup (click on the task in the list)
