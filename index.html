About

checklist provides an interface to Google Tasks™ via a very simple and intuitive Chrome extension. Lists of tasks and the ability to quickly add a task are a keyboard shortcut away.

  • fast access to tasks from Chrome
  • ability to add a page, or text from a page, as a task directly via right-click menu and keyboard shortcuts
  • great for use with gmail to make a task out of an email; adding a page as a task stores the URL of that page for quick access later
  • support for multiple lists and consolidated view of all lists

checklist browser extension communicates directly with Google Tasks™ via Google APIs and does not otherwise persist or store your data anywhere else.
